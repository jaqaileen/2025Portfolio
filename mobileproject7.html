In Praise of Shadows

Creative Coding | Web Design

Inspired by Jun'ichirō Tanizaki's essay of the same name, this website is a desktop-friendly creative coding project. This work explores the intersection of typography and interaction, using light, motion, and contrast to echo Tanizaki’s meditations on shadow and subtlety.

The project features a typographic dialogue between the contemporary sans-serif Buljirya and the classical serif Cardo, inviting users to engage with text in a space shaped by quiet tension and visual play.
